# Analytics Component Archive

This folder contains several archived/prototype analytics components that are intentionally preserved as Medica roadmap assets.

These components are currently not mounted in the active analytics dashboard. They should not be deleted only because they have zero imports or do not appear in the current runtime bundle.

They may be reused later for:

- readiness explainability
- mastery breakdowns
- progress trends
- clinical reasoning feedback
- mistake diagnosis
- study prescription UX
- benchmark/cohort intelligence
- flashcard recommendations

Preserved components:

- `AccuracyOverview.jsx`
- `BenchmarkIntelligence.jsx`
- `ClinicalReasoningPattern.jsx`
- `FlashcardRecommendationSummary.jsx`
- `MedicaScoreCard.jsx`
- `MistakeDiagnosis.jsx`
- `NextSessionRecommendation.jsx`
- `ProgressGainsChart.jsx`
- `ProgressTrends.jsx`
- `StudyPrescription.jsx`
- `SubjectBreakdown.jsx`
- `SystemBreakdown.jsx`
- `TopicBreakdown.jsx`

Treat these files as future feature prototypes unless a product/engineering review explicitly marks them obsolete.
